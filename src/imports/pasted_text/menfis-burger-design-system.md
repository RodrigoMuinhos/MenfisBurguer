# PROMPT FIGMA AI — MENFI'S BURGER OS

Crie o design completo de um sistema SaaS para gestão e operação da hamburgueria Menfi's Burger.

O objetivo é desenvolver primeiro toda a experiência de produto (UX/UI) antes da implementação.

O resultado deve parecer um produto comercial pronto para venda, com aparência profissional, moderna, escalável e extremamente simples de usar.

---

# Contexto do Negócio

Menfi's Burger é uma hamburgueria com operação enxuta.

Atualmente existe apenas:

### Menfi's Burger

* Pão Brioche
* Hambúrguer Artesanal 100g
* Cebola Caramelizada
* Alface Americana
* Queijo Assado
* Molho Menfi's

Preço:
R$ 19,90

---

### Combo Menfi's

Inclui:

* Menfi's Burger
* Coca-Cola 350ml
* Batata Rústica 250g

Preço:
R$ 34,90

---

Apesar de possuir poucos produtos, o sistema deve oferecer gestão profissional.

---

# Objetivo do Sistema

Criar um ecossistema integrado composto por:

1. Totem de Autoatendimento
2. Web App do Cliente
3. Delivery Próprio
4. KDS (Kitchen Display System)
5. ERP Administrativo
6. Controle de Estoque
